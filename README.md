# Slack Connect App

A full-stack web application that integrates with Slack using OAuth 2.0. Built using React, Node.js, and TypeScript.



## Setup

### Backend
1. Create `.env` file with Slack credentials.
2. Run:
```bash
cd backend
npm install
npm run dev
```

### Frontend
1. Run:
```bash
cd frontend
npm install
npm start
```
